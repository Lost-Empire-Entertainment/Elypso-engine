# External libraries

This project relies on several external libraries, they are not shipped in its repository and are found [here](https://github.com/greeenlaser/external-shared).
These libraries are used for various purposes, their licenses are included in their folder and this repository license does not apply to them.

## Owned by Lost Empire Entertainment

| Library                                                 | Use case                                                       |
|---------------------------------------------------------|----------------------------------------------------------------|
| [KalaWindow](https://github.com/KalaKit/KalaWindow)     | Windowing, input, crash handler and Vulkan context. |
| [KalaGraphics](https://github.com/KalaKit/KalaGraphics) | Cameras, 3D models, UI, light sources, particle effects and post processing. |
| [KalaPhysics](https://github.com/KalaKit/KalaPhysics)   | Physics TBD. |
| [KalaAudio](https://github.com/KalaKit/KalaAudio)       | Audio playback and audio listener. |
| [KalaLua](https://github.com/KalaKit/KalaLua)           | Wrapper library for Lua. |
| [KalaHeaders](https://github.com/KalaKit/KalaHeaders)   | Lightweight, independent header-only scripts for various uses. |
